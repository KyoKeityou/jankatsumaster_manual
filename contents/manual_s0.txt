<summary class="linkfont_color">1.概要</summary> 
雀活マスターは麻雀プロの方々のゲスト情報（対局予定、イベント、配信など）を検索・フォローできるアプリです。
<br/>
<br/>
<img src="./images/logo_toumei.png" border="1" style="max-width: 40%; height: auto;" /> 
<br/>
<br/>
<b>主な機能：</b>
<br/>
・麻雀プロの検索、お気に入り登録（フォロー）
<br/>
・カレンダーで麻雀プロのスケジュール確認
<br/>
・スケジュール更新時のプッシュ通知
<br/>
・日単位で麻雀プロの方々のゲスト情報を一覧で確認
<br/>
・麻雀プロのツイッター（X）へのリンク
<br/>
・便利ツール（電卓・サイコロ）
<br/>
・大会情報のお知らせ
<br/>
<br/>
