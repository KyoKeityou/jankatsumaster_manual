<summary class="linkfont_color">5.お気に入り一覧</summary> 
お気に入り登録された麻雀プロ／雀荘／麻雀BARが一覧で表示されます。
<br/>
<br/>
<img src="./images/favorite_list.jpg" border="1" width="318" height="689" /> 
<br/>
<br/>
<b>使い方：</b>
<br/>
・未設定 - タップすると、カレンダーに表示する背景色の新規設定ができます。
<br/>
・設定済 - タップすると、カレンダーに表示する背景色の変更ができます。
<br/>
・解除⛔️ - タップすると、お気に入り登録を解除し、一覧から削除されます。
<br/>
<br/>
・「お気に入り登録／解除」をタップすると、検索画面に遷移します。
<br/>
