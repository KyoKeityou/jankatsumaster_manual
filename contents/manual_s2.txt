<summary class="linkfont_color">3.カレンダー</summary> 
お気に入り登録済の麻雀プロのスケジュールを月間で確認できます。
<br/>
<br/>
<img src="./images/calendar_month.jpg" border="1" width="318" height="689" /> 
<br/>
<br/>
<b>使い方：</b>
<br/>
・「<」 - タップすると、前の月へ遷移します。
<br/>
・「>」 - タップすると、次の月へ遷移します。
<br/>
・日付に麻雀プロの名前が表示されている場合、その日は該当プロの予定があることを示しています。
<br/>
・日付をタップするとその日の予定を表示します。
<br/>
