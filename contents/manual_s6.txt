<details>
<summary class="linkfont_color">7.ゲスト一覧</summary> 
日単位で麻雀プロの方々のゲスト情報を一覧で確認できます。
<br/>
<br/>
<img src="./images/guest_list.jpg" border="1" width="318" height="689" /> 
<br/>
<br/>
<b>使い方：</b>
<br/>
・カレンダーアイコン - タップすると、日付の選択ができます。
<br/>
・「<」 - タップすると、前の月へ遷移します。
<br/>
・「>」 - タップすると、次の月へ遷移します。
<br/>
<br/>
・麻雀プロの氏名をタップすると、その方の月スケジュールを表示します。
<br/>
<br/>
<img src="./images/guest_calendar_month.jpg" border="1" width="318" height="689" /> 
<br/>
<br/>
・麻雀プロのプロフィール写真をタップすると、X（ツイッター）アプリを開き、その方のXにあるプロフィールを表示します。
<br/>
<br/>
